# Flask API с аутентификацией, проектами и API-ключами

Полноценный веб-сервис для регистрации пользователей, управления проектами и генерации API-ключей для доступа из сторонних агентов.

## Состав проекта

- **backend/** – серверная часть на Flask (Python)
- **frontend/** – клиентская часть на чистом HTML/CSS/JavaScript (открывается в браузере)
- **agent/** – консольный агент на Python для демонстрации доступа по API-ключу
