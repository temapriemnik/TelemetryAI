# Telemetry AI

# О проекте
проект автоматизации мониторинг. SAAS. нацелен на легаси системы и docker compose/swarm инфраструктуру

# Архитектура
Сатилитная архитектура - модульный монолит - backend + сатилиты agent, сервис модели

# Артем
Разработка бэкэнда - логин, вход, управление проектами, rool based детекция логов,

инжекция логов в брокера сообщений, и управление событиями и отображением логов. Разработка агента для работы с SMTP,

разработка агента сбора логов.

# Артур

Разработка ui интерфейса системы. html css tailwind css js jquiry

Работа с JWT токенами, websocket

# Егор

Работа с датасетом, анализ и обработка данных, обучение модели машинного обучения. оптимизация распределенной работы с google colab
